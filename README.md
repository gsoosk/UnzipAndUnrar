# UnzipAndUnrar
A simple shell script to unzip and unrar all archive files in subrepositories recursively.

## Dependencies
```
brew install unrar
```

## How it works ? 
```
chmod 755 unzipAndUnrar.sh
./unzipAndUnrar.sh [Directory] [-r]? 
```
NOTE : _-r is for removing archive files after extracting_
